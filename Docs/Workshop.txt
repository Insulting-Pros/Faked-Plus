This mutator is an advanced version of Faked Players. Allows to change fakes, zeds health, switch slomo on-off and many other things directly from the game.

Custom mode ('Chosen amount of fakes') allows a bit more than others. You can reserve player slots and use more than 5 fakes.

[h2][b]Whitelist Status[/b][/h2]
[list]
[*] Dedicated server: all clients will level up, get achievs.
[*] Listened server: except the host itself, all clients will level up, get achievs.
[*] Solo: greylisted. No regrets.
[*] Using Custom mod with tons of fakes will disable level up, achievs.
[/list]

[h2][b]Console Commands[/b][/h2]
Commands are named pretty obvious so I'm listing them here for acquaintance, without many detailes.
If you want to find out what cmd does what - type [b]'mutate HELP'[/b] and [b]'mutate HELP <cmdname>'[/b] during a game. If you want to check what variables are currently used type [b]'mutate STATUS'[/b].

[h2][b]Commands with ON-OFF switch:[/b][/h2]
[list]
[*] LOCK
[*] SPEC, SPECS, SPECTATOR, SPECTATORS
[*] DRAMA, SLOMO
[*] ADMINONLY, ADMIN
[*] SOLO
[/list]

[h2][b]Commands with Int values:[/b][/h2]
[list]
[*] FAKED, FAKE, FAKES
[*] PLAYER, PLAYERS, SLOT
[*] HEALTH, HP
[*] SPEC, SPECS, SPECTATOR, SPECTATORS
[*] ReservedSlots, RS
[*] ConfigFakes, CF
[/list]

[/list]
